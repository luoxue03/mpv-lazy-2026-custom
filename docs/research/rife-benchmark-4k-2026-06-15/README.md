## 推荐解读

- 如果追求满高 4K 不降处理高度，优先看 `H_Pre=2160` 中 `median_fps >= 55` 的组合。
- 如果接受处理高度降到 `1920/1608/1440`，多数模型都有明显余量。
- `Turbo 0` 多数偏慢，不适合满高 4K 实时播放。
- `4.6 + Turbo 1 + flow_scale=0.5/0.25` 确实可跑，但这是特定实现路径；不要扩展到 Turbo 2。
